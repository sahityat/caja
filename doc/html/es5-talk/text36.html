Nonstrict global object leakage

function Point(x,y) {

this.x = +x;

this.y = +y;

}

var pt = Point(3,5); //Oops. Forgot “new”

Nonstrict: {window.x = 3; window.y = 5;}

Strict: {undefined.x = 3; undefined.y = 5;}

Safer. And happens to throw quickly in this case
